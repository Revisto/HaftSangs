HaftSangs is an online multiplayer game inspired by the traditional Iranian game "هفت سنگ" (Haft Sang). Players shoot a ball to 7 stones (haft sangs) while the opponent tries to protect and stop the stones using obstacles. This game was developed for the "AUT Gamecraft" game jam.
